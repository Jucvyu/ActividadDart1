## Descripción

Esta actividad consiste en desarrollar una aplicación de consola en Dart que solicita información de un aprendiz, calcula el promedio de tres notas y determina si el estudiante aprobó o no el curso.

---

## Objetivo

Aplicar los conceptos básicos del lenguaje Dart mediante el desarrollo de una aplicación de consola que permita el manejo de datos, operaciones y estructuras de control.

---

## Temas trabajados

- Entrada y salida de datos.
- Declaración y uso de variables.
- Tipos de datos.
- Operadores aritméticos.
- Operadores relacionales.
- Estructuras condicionales (`if` y `else`).
- Conversión de datos.
- Cálculo de promedios.

---

## Requisitos

- Tener instalado el SDK de Dart.
- Visual Studio Code (opcional, pero recomendado).

---

## Instrucciones de ejecución

1. Abrir el proyecto en Visual Studio Code.
2. Abrir una terminal en la carpeta del proyecto.
3. Ejecutar el siguiente comando:

```bash
dart run
```
